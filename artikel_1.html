Pernahkah Anda membuat aplikasi GIS (Geographic Information System) atau aplikasi pemetaan berbasis web? 


Jika belum, mungkin Anda bisa memulai dengan memahami konsep GIS secara umum di situs ESRI.

Jika sudah, mungkin anda pernah mendengar library Javascript yang bernama Leaflet.

 

Leaflet merupakan library yang dibuat oleh Vladimir Agafonkin dan beberapa kontributor lainnya yang dirilis pada 30 Juli 2012 dengan versi 0.4. Leaflet sampai sekarang juga masih terus dikembangkan. Pada saat tulisan ini dibuat, versi terakhir yang dirilis adalah 1.0 pada 27 September 2016.

Leaflet is the leading open-source JavaScript library for mobile-friendly interactive maps

 Leaflet bertujuan untuk membantu para developer yang ingin membuat aplikasi GIS yang interaktif dan bisa bekerja secara optimal pada perangkat-perangkat desktop maupun mobile.

Leaflet juga menyediakan beberapa plugin seperti  

  • Basemap Providers
  • Vector Tiles
  • Markers
  • Interactive Pan/Zoom
  • Geolocation
  • Search and Popups
  • Geocoding
  • dan lain-lain

Selain itu, Leaflet juga memiliki dokumentasi yang lengkap tentang penggunaan, konfigurasi, event handling, dan masih banyak lagi hal lainnya yang akan kita pelajari pada bagian-bagian selanjutnya.

Pada awal ini, ada beberapa hal yang harus disiapkan, yaitu

  1. Web Server yang udah terinstall, anda bisa memakai XAMPP, LAMP, atau sejenisnya
  2. Editor untuk membuat kode yang akan kita eksekusi 
  3. Browser yang akan digunakan untuk menampilkan peta

Langkah-langkah yang dilakukan adalah:

1. Silakan signup di MapBox

2. Buat API access token MapBox di bagian Tokens atau mengklik link ini

3. Generate New Token

4. Masukkan nama token

5. Hasil dari generate token seperti berikut (yang ditandai merah maroon)

6. Buatlah sebuah file di folder yang bisa dikases untuk web (XAMPP biasanya di folder htdocs) bernama gis.php

 

<!DOCTYPE html>
<html>
<head>	
	<title>Belajar Leaflet</title>

	<meta charset="utf-8" />
	<meta name="viewport" content="width=device-width, initial-scale=1.0">

	<link rel="stylesheet" href="https://unpkg.com/leaflet@1.0.1/dist/leaflet.css" />
	<script src="https://unpkg.com/leaflet@1.0.1/dist/leaflet.js"></script>

</head>
<body>

	<div id="peta" style="width: 600px; height: 400px;"></div>
	<script src="app.js"></script>

</body>
</html>

7. Buatlah sebuah file di tempat yang sama bernama app.js. Isikan MAPBOX_ACCESS_TOKEN dengan token yang didapat pada langkah 5 

var mymap = L.map('peta').setView([-6.231, 106.865], 17);

L.tileLayer('https://api.tiles.mapbox.com/v4/{id}/{z}/{x}/{y}.png?access_token=MAPBOX_ACCESS_TOKEN', {
    maxZoom: 18,
    attribution: 'Map data &copy; <a href="http://openstreetmap.org">OpenStreetMap</a> contributors, ' +
        '<a href="http://creativecommons.org/licenses/by-sa/2.0/">CC-BY-SA</a>, ' +
        'Imagery © <a href="http://mapbox.com">Mapbox</a>',
    id: 'mapbox.streets'
}).addTo(mymap);

8. Silakan akses file gis.php menggunakan browser kesukaan anda. Di mesin saya, url yang saya akses adalah http://localhost/gis/gis.php 

9. Hasilnya seperti berikut

10. Jika sudah terlihat seperti pada gambar di atas, berarti aplikasi awal sudah jalan

 

Pada file gis.php,  <link rel="stylesheet" href="https://unpkg.com/leaflet@1.0.1/dist/leaflet.css" /> dan  <script src="https://unpkg.com/leaflet@1.0.1/dist/leaflet.js"></script> merupakan 2 file yang diambil dari CDN memuat css dan js Leaflet yang akan digunakan untuk membuat peta. <div id="peta" style="width: 600px; height: 400px;"></div> merupakan tempat kita akan menempatkan peta.  <script src="app.js"></script>  memanggil app.js.

 

Pada file app.js,  

var mymap = L.map('peta').setView([-6.231, 106.865], 17);

menunjukkan inisialisasi peta bernama mymap pada div dengan class="peta". [-6.231, 106.865]  merupakan koordinat dari titik tengah dari peta pada saat pertama di-load. 17 adalah zoom pada saat pertama di-load.

tileLayer merupakan menyimpan peta dasar yang dipakai. Pada parameter pertama  menunjukkan penggunaan tiles dari Mapbox. Pada parameter kedua, terdapat beberapa variabel. maxZoom menunjukkan maksimal zoom yang bisa dilakukan. attribution menunjukkan atribut atau keterangan yang tertera pada peta (bawah kanan layar). id merupakan penanda yang disediakan oleh MapBox. Selain mapbox.street, MapBox juga menyediakan pilihan lain (silakan dicoba-coba sesuai dengan kebutuhan), seperti

  • mapbox.streets
  • mapbox.light
  • mapbox.dark
  • mapbox.satellite
  • mapbox.streets-satellite
  • mapbox.wheatpaste
  • mapbox.streets-basic
  • mapbox.comic
  • mapbox.outdoors
  • mapbox.run-bike-hike
  • mapbox.pencil
  • mapbox.pirates
  • mapbox.emerald
  • mapbox.high-contrast

addTo(mymap) artinya tileLayer tersebut dimasukkan ke dalam peta yang telah didefinisikan sebelumnya sehingga peta bisa tampil.
